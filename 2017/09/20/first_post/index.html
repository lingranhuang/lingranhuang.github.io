前几天收到了域名快过期的邮件，上来看才发现写了两篇，因为人比较懒。决定好好整理一下，也没什么可以说的事情，但是最近好好反醒了一下，还是觉得记录一些东西，不管是技术还是想法。

主要认识到几个问题:

  1. 这一段时间在调研OLAP, 深深感觉到自己还是太弱了，这样下去怎么可能去做自己喜欢的事件。
  2. 学习不下去，每天挤出一点时间去看书，但碎片化的学习效率实在是太低了，本来时间就很短，却都几乎花在了回忆昨天所记的东西，导致效率低下，巨大的挫败感严重降低学习兴趣。
  3. 对于自己定的一些计划和目标，总是太容易放弃，当然大多以时间为借口，因为放弃的成本太低。
  4. 身体越来越差，相比一年前，胖了有8公斤左右，但是下半身力量没提升反而有点衰弱，现在去跑步跑完一半疼得受不了，明显感觉到在衰老。
  5. 熬夜太多，状态越来越差，白天完全没办法专注去做一件事件（晚上状态真是极好，相比于白天安静许多）。

当然，最大的原因是没时间了，希望做完更多的事:
